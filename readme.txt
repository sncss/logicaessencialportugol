# Exercícios de lógica de programação em portugol

Este repositório contem exercícios de lógica de programação de nível iniciante em Portugol. O objetivo é registrar o meu progresso nos estudos de programação.

## Começando

Para começar, caso queira abrir os exercícios em sua máquina, será necessário o download do software Portugol Studio no seguinte link: https://univali-lite.github.io/Portugol-Studio/.

- Faça o download dos arquivos .por;
- Abra no Portugol Studio e explore cada arquivo de exercício para praticar.
